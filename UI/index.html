Input Scrape :

This is the area to add a recurring scrape to the application.
Once you have decided which URL you would like to scrape, add the address directly into the 'target URL' area (copy paste works GREAT!)
Add the URL name directly to the Target URL
decide how often you would like to scrape. currently we have 3 options : daily, weekly, and monthly. web gnome will default to daily.
This will add a process in the background to scrape accordingly. (additional scrapes will not add more processes.)
Lastly, the scrape recursively button (if you select it) will go through each connected website, and scrape those as well. This is great if you want to track the entire infrastructure of a website. Be careful though : Large websites (such as wikipedia) may take a large time to run. In the future, may implement a maximum depth to scrape.

Saved Scrapes:

This is here to see your scrapes, and (in the future) modify the paramters of your scrapes.
Key :
Number: for your convience tracking them. No functional purpose
Website name: the name of the Website (and sometimes subpage)
Scrape Frequency : how often web gnome will run a scrape on this Website
Scraped previously : determines if web gnome should compare scrapes to find differences
Scrape recursively : asking if web gnome should scrape any connected web-pages
